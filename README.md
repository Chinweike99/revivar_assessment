# REVIVAR ASSESSMENT
## Thank You Card Generator
### A beautiful, responsive web application that allows users to create personalized thank you cards using stunning images from Unsplash. Built with modern technologies including TypeScript, Next.js, Tailwind CSS, and GSAP for smooth animations.

## Features
### Core Functionality
    - Random Image Selection: Automatically fetches 4 random portrait-oriented images from Unsplash
    - Image Search: Real-time search functionality with pagination for finding perfect images
    - Personalized Cards: Overlay "Thank You" text at the top and user's name at the bottom
    - Downloadable Cards: Generate and download high-quality PNG images (4:5 aspect ratio)
    - Responsive Design: Optimized for all device sizes

### Customization Options
    - Font Selection: Choose from 6 different font families (Arial, Georgia, Times New Roman, Helvetica, Verdana, Courier New)
    - Color Picker: 8 pre-selected text colors with visual color swatches
    - Live Preview: Real-time preview of card design changes
    - Professional Layout: Clean, modern interface with smooth animations

### Technical Excellence
    - TypeScript: Full type safety with zero any types
    - GSAP Animations: Smooth, professional animations for image loading and card creation
    - Tailwind CSS: Utility-first styling with consistent design system
    - Next.js 16: Latest React framework with optimal performance
    - Responsive: Mobile-first design approach
